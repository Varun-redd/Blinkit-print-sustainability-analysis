## Overview
This project analyzes paper waste generated by print orders and evaluates the impact of introducing an **Eco Mode** for single-page prints.  
The goal is to reduce unnecessary paper usage while delivering measurable **cost and sustainability benefits**.

---

## Problem Statement

## Key Insights
- ~45% of orders generate avoidable paper waste
- Single-page orders contribute disproportionately to total waste
- Partial Eco Mode adoption delivers significant cost and waste reduction
- Sustainability gains align directly with cost efficiency
